### Multiple Solutions Included

| Method                    | Description                              | Recommended |
|--------------------------|------------------------------------------|-----------|
| Basic Loop               | Manual character-by-character replacement | Good for learning |
| `str.replace()`          | Simple and readable one-liner            | **Best for most cases** |
| `str.translate()`        | High-performance method                  | Best for large data |
| Biopython                | Using `Seq.transcribe()`                 | Biological context |
